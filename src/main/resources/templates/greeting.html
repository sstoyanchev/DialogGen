Form-based dialog generation

  1. Download and unzip Opendial system.
    You will need java 8 jdk to run it.
    Make sure your JAVA_HOME environment variable points to java 8 (e.g./Library/Java/JavaVirtualMachines/jdk1.8.0_45.jdk/Contents/Home)
  2. Run it with:
    cd opendial
    scripts/opendial (text only);
    scripts/opendialWithspeech (with speech)
  3. Open the web web interface and click on "Load Hotel Example"
  4. Review the form and click 'submit' (downloads opendial.xml)
  5. In the Opendial window, click Domain | openfile and choose the downloaded opendial.xml
  6. Try filling out the form using dialog (the system is user-driven and fields can be filled in a single or in multiple utterances and in any order).
    To reset the form for a new dialog click on Interaction | Reset.
    Please keep track of the number of dialog paths you attempt and the number that are successful.
    A successful form completion results in filled-in form window displayed.
    • Try filling out the form by specifying all fields in a single utterance.
    • Try filling the form by specifying one field at a time.
    • Please try three other dialog paths for filling out the form.
  7. Reset the form and try filling out other examples (e.g. use Luis interface, add fields, change questions)
  8. Proceed to questionnaire
